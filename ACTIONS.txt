Claude Code DevTools — 可用操作列表
=====================================

CC 在回复中使用以下标签来操作网页和文件，面板会自动解析执行并回传结果。

[ACTION:eval]JavaScript代码[/ACTION]
    在页面上执行 JS 并获取返回值
    例: [ACTION:eval]document.querySelectorAll('a').length[/ACTION]

[ACTION:dom]CSS选择器[/ACTION]
    获取匹配元素的 outerHTML（仅第一个）
    例: [ACTION:dom]#main[/ACTION]

[ACTION:dom:all]CSS选择器[/ACTION]
    获取所有匹配元素的 outerHTML
    例: [ACTION:dom:all].item[/ACTION]

[ACTION:text]CSS选择器[/ACTION]
    获取元素的可见文本内容
    例: [ACTION:text]h1[/ACTION]

[ACTION:console][/ACTION]
    获取最近的控制台日志（最多200条）

[ACTION:network][/ACTION]
    获取最近20条网络请求（需打开 Network 面板）

[ACTION:title][/ACTION]
    获取页面标题

[ACTION:url][/ACTION]
    获取当前页面 URL

[ACTION:copy]要复制的内容[/ACTION]
    将内容复制到系统剪贴板
    例: [ACTION:copy]console.log('hello')[/ACTION]

[ACTION:file:list]glob模式[/ACTION]
    列出 Bridge Server 允许目录内的本地项目文件
    自动过滤 .git、node_modules、dist、build、__pycache__
    例: [ACTION:file:list]*countr*[/ACTION]

[ACTION:file:read]文件路径[/ACTION]
    读取 Bridge Server 允许目录内的本地项目文件
    例: [ACTION:file:read]src/data/countries.json[/ACTION]

[ACTION:save]文件路径
文件内容（从下一行开始）
[/ACTION]
    将内容写入 Bridge Server 允许的目录，自动创建目录
    默认允许目录是 Bridge Server 启动时的工作目录
    可用 CC_DEVTOOLS_WRITE_ROOT 指定允许目录
    例: [ACTION:save]index.html
        <!DOCTYPE html>
        <html>...</html>
        [/ACTION]
