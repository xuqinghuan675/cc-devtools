# Country Selector Loop Demo

Bundled demo for `cc-devtools-demo` and the **Frontend Loop** workflow.

Prompt:

```text
Add Singapore to the country selector. Use the local JSON file, then select it and verify it in the page.
```

Run page server:

```bash
python -m http.server 5173
```

Run bridge from the same directory:

```bash
cc-devtools
```
